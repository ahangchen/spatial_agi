## 📚 详细技术分析

### Orster: 正交空间-时间分布迁移的完整技术细节

**核心架构**: STD-4D Diffusion Model

\`\`\`
输入: 文本描述 y
输出: 4D内容 X̂^{4D} = {X̂_t}_{t=1}^T

流程:
1. VAE编码: Z_0^{4D} = VAE_enc(X^{4D})
2. 前向扩散: Z_t^{4D} = √(ᾱ_t)Z_0^{4D} + √(1-ᾱ_t)ε, ε~N(0,I)
3. 解耦: Z_t^S = Disentangle_spatial(Z_t^{4D}), Z_t^T = Disentangle_temporal(Z_t^{4D})
4. 去噪: Z_{t-1}^S = ε_θ^S(Z_t^S, t), Z_{t-1}^T = ε_θ^T(Z_t^T, t)
5. 融合: Z_{t-1}^{4D} = FFN(Z_{t-1}^S, Z_{t-1}^T | y)
6. VAE解码: X̂^{4D} = VAE_dec(Z_0^{4D})
\`\`\`

**Orster机制**:

联合时空分布高斯核:
\`\`\`
κ(f_s, f_t) = exp(-1/2 * (||f_s - g_s||²/σ_s² + ||f_t - g_t||²/σ_t² + 2α⟨f_s - g_s, f_t - g_t⟩/σ_st²))
\`\`\`

**ST-HexPlane**:

4D特征分解:
\`\`\`
F^{4D}(x,y,z,t) ≈ ∑_{i=1}^R P_{xy}^i(x,y) ⊗ P_{zt}^i(z,t) ⊗ P_{xz}^i(x,z) ⊗ P_{yt}^i(y,t)
\`\`\`
