## 摘要

### 研究问题

随着无线生态系统的快速扩展,包括地面和空中域的设备密度不断增加,3D频谱环境变得日益拥挤。传统的静态频谱分配模型已无法满足现代连接需求,频谱稀缺问题日益严重。认知无线电技术通过智能频谱管理策略(如动态频谱共享)来应对这一挑战,而这需要全面的实时频谱占用感知。

3D无线环境地图(Radio Environment Map, REM)是核心使能技术,它是一个空间数据库,表征特定坐标和高度处的接收功率水平(如参考信号接收功率RSRP)。构建密集、高保真的3D REM是优化功率控制、频率分配并确保干扰缓解的基础要求。

### 核心挑战

虽然无人机(UAV)提供高机动性3D感知能力,但REM精度面临多重挑战:

1. **动态飞行行为**: UAV速度和方向的波动引入测量不一致性
2. **机身结构影响**: 机身本身会影响机载天线的辐射特性
3. **测量噪声**: 来自旋翼和机载电子设备的电磁干扰
4. **空间相关性**: 飞行路径严重影响测量的空间相关性
5. **天线模式失真**: 实际部署中天线模式与暗室测量结果存在显著差异

### 主要贡献

本研究使用来自固定基站塔和地面车辆源的实测数据,对各种高度下的REM重建进行全面分析:

1. **方法评估**: 评估了Kriging(简单、普通和trans-Gaussian)、矩阵补全和高斯过程回归(GPR)等多种重建方法
2. **鲁棒性发现**: 简单Kriging和GPR在极端样本稀疏情况下更具鲁棒性
3. **深阴影框架**: 提出特殊框架,通过将REM分解为平滑和深阴影空间组件来提高深阴影区域的重建精度
4. **高度影响**: 揭示UAV高度对精度的影响遵循三相趋势
5. **带宽影响**: 证明性能随频谱带宽增加而改善
6. **仰角影响**: 阴影衰落方差呈现非单调趋势,在极低和中高仰角处达到峰值
7. **天线校准**: 通过现场测量校准天线模式显著增强REM重建精度

## 研究背景与动机

### 频谱稀缺与认知无线电

无线生态系统的快速扩张引发了频谱稀缺的关键问题:

- **设备多样性**: femtocell、IoT设备、5G车辆、UAV等
- **3D频谱复杂性**: 地面和空中域的密集化
- **静态分配失效**: 传统静态分配模型无法满足现代连接需求

认知无线电技术通过动态频谱共享解决这一问题,其核心是构建全面的3D REM。

### REM重建方法演进

文献中的REM重建方法可分为三类:

#### 1. 直接重建方法

将问题视为通用数据插值或张量补全任务,不考虑无线电传播的底层物理:

- **经典插值**: Kriging、反距离加权(IDW)、高斯过程回归(GPR)
- **深度学习**: U-Net、自编码器、GAN、扩散模型
- **矩阵补全**: 利用低秩特性预测未知网格值

#### 2. 间接重建方法

利用无线电传播物理,假设确定性传播模型:

- **参数估计**: 路径损耗指数、LoS概率、阴影衰落方差
- **源定位**: 正交匹配追踪(OMP)、稀疏贝叶斯学习(SBL)
- **天线模式调优**: 非线性优化

#### 3. 混合方法

结合直接和间接方法的优势:

- **两阶段建模**: 确定性路径损失 + 数据驱动阴影衰落
- **环境语义**: 几何传播模型 + 机器学习预测
- **残差估计**: 物理模型 + Kriging残差

### UAV辅助感知的挑战

尽管UAV辅助感知具有独特优势,但仍面临平台特定挑战:

1. **噪声源多样性**:
   - 旋翼电磁干扰
   - 机载电子设备噪声
   - 环境阴影和多径衰落

2. **飞行路径影响**:
   - 相反方向穿越相同轨迹产生显著信号变化
   - UAV方向和航向变化
   - 速度波动

3. **天线模式失真**:
   - 机身反射和散射
   - UAV机翼阴影
   - 增益畸变

### 研究空白

现有研究的不足之处:

- **合成数据验证**: 大多数方法在统计或射线追踪合成数据上验证
- **高度依赖性**: REM重建误差与高度、带宽、仰角的关系研究不足
- **天线校准**: UAV特定天线模式校准对3D REM重建精度的影响探索有限

## 核心方法

### 系统模型

#### 环境配置

**节点设置**:
- **地面节点**: 固定基站(BS)或地面站(GS),发射无线电信号
- **UAV节点**: 携带便携式频谱传感器,收集RSRP测量值

**天线配置**:
- **地面天线**: RM-WB1-DN天线,安装在顶部
- **UAV天线**: SA-1400-5900宽带天线,向下定向,安装在水平杆上
- **分离设计**: 天线固定在水平杆上,与UAV/GS其他组件分离

**环境假设**:
- 农村环境
- 视距(LoS) + 地面反射射线
- 双射线传播模型

#### 坐标系统

**3D坐标定义**:
- 地面站天线: l_gs = (ψ_gs, ζ_gs, h_gs)
- UAV天线: l_uav = (ψ_uav, ζ_uav, h_uav)
- 其中ψ为纬度,ζ为经度,h为高度

**距离计算**:

1. **水平距离** (考虑地球曲率):
   ```
   d_h(l_gs, l_uav) = R_e · arccos(sin ψ_uav sin ψ_gs 
                      + cos ψ_uav cos ψ_gs cos(ζ_gs - ζ_uav))
   ```
   其中R_e为地球半径

2. **垂直距离**:
   ```
   d_v(l_gs, l_uav) = |h_gs - h_uav|
   ```

3. **3D距离**:
   ```
   d_3D = √(d_h² + d_v²)
   ```

### 两射线路径损耗模型(TRPL)

TRPL模型计算信号衰减因子,考虑LoS和地面反射路径:

```
A_trpl(l_gs, l_uav) = (λ_wave / 4π)² |G_gs(φ_t, θ_t) G_uav(φ_r, θ_r) / d_3D 
                       + Γ(θ_ref) G_gs(φ_t1, θ_t1) G_uav(φ_r1, θ_r1) e^(-jΔτ) / (d_1 + d_2)|²
```

**关键参数**:
- **A_trpl**: 信号衰减因子(接收功率/发射功率)
- **G_gs(·), G_uav(·)**: GS和UAV天线增益
- **θ_t, θ_t1, θ_r, θ_r1**: 发射和接收仰角
- **λ_wave**: 波长
- **Δτ**: 两路径间相位延迟
- **Γ(θ_ref)**: 地面反射系数

**路径损耗(dB)**:
```
PL_trpl^(dB) = -10 log₁₀(A_trpl)
```

**估计接收功率**:
```
r̂_trpl^(dB) = P_Tx^(dB) - PL_trpl^(dB)
```

### 阴影衰落的空间相关性

阴影衰落(SF)遵循空间相关的随机过程:

**空间相关性定义**:
```
R_w(l_i, l_j) = E[w(l_i) w(l_j)] / σ_w²
```

**相关性模型**(双指数分布):
```
R_w(l_i, l_j) = e^(-q d_v) [a e^(-p₁ d_h) + (1-a) e^(-p₂ d_h)]
```

**参数**:
- **a, p₁, p₂, q**: 从实测数据估计的常数
- **d_v, d_h**: 垂直和水平分离距离

该模型捕捉了大尺度衰落(由显著物理障碍引起)在3D环境中的平滑变化特性。

### 问题设置

**目标**: 评估不同重建方法在一致框架下的性能,并研究物理参数(高度、带宽、仰角)对REM精度的影响。

**设置**:
1. **固定高度测量**: 在常量高度H进行测量活动ℰ
2. **观测数据**: n个观测信号功率数据点 P_Rx^(dB)(l_i), i ∈ {1,...,n}
3. **稀疏采样**: 从n个测量中随机选择M个点作为重建的稀疏样本
4. **稀疏度范围**: M从10到250,分析不同数据稀疏度下的性能
5. **测试集**: 剩余(n-M)个点作为ground-truth测试集

**已知信息**:
- 发射机位置
- 隔离天线的暗室天线模式
- 发射功率

**训练设置**:
- **训练数据集**: 在高度H'进行的单独UAV活动ℰ',包含n'个测量(n' > M)
- **用途**: 估计空间相关性参数或优化深度学习模型权重
- **推理数据**: 来自目标实验ℰ的有限M个样本
- **空间分离**: 确保ℰ和ℰ'之间的严格空间分离(垂直分离≥20m或水平轨迹不相交)

## 算法流程

### 混合重建框架

**核心思想**: 结合确定性无线电传播模型(大尺度路径损耗预测)和数据驱动方法(随机成分估计)。

**重建接收功率**:
```
r̂^(dB) = r̂_trpl^(dB) + ŵ
```

其中:
- **r̂_trpl^(dB)**: TRPL模型预测的接收功率
- **ŵ**: 数据驱动技术预测的阴影衰落(SF)

**阴影衰落提取**:
```
w_trpl = P_Rx^(dB) - r̂_trpl^(dB) = Z
```

这些已知的Z值作为通过空间相关性特性预测未采样位置SF的主要输入。

### 方法1: 普通Kriging (OK)

**原理**: 使用已知测量的线性组合预测新位置l₀的信号Ẑ(l₀),最小化平方估计误差,同时确保权重和为1。

**优化问题**:
```
min_{w₁,...,w_M} E[(Ẑ(l₀) - Z(l₀))²]

s.t. Ẑ(l₀) = Σᵢ₌₁ᴹ wᵢ Z(lᵢ)
     Σᵢ₌₁ᴹ wᵢ = 1
```

**求解**: 使用拉格朗日乘子法,转化为线性方程组:
```
[γ(l₁,l₁)  ...  γ(l₁,l_M)  1] [w₁]   [γ(l₀,l₁)]
[  ⋮       ⋱     ⋮        ⋮ ] [⋮ ] = [  ⋮    ]
[γ(l_M,l₁) ... γ(l_M,l_M) 1] [w_M]   [γ(l₀,l_M)]
[  1       ...    1        0] [μ ]   [   1   ]
```

**半变异函数**:
```
γ(l_i, l_j) = Var(Z(l_i) - Z(l_j)) = σ_Z² [1 - R_z(l_i, l_j)]
```

**预测MSE**:
```
σ_ok²(l₀) = Σᵢ₌₁ᴹ wᵢ γ(l₀, lᵢ) + μ
```

**特性**: 最佳线性无偏估计器(BLUE),通过假设局部恒定但未知的均值并施加权重和为1的约束来确保无偏性。

### 方法2: 简单Kriging (SK)

**原理**: 与OK类似,但不施加权重和约束,假设过程Z具有恒定均值m_Z(对于SF通常接近零)。

**优化问题**:
```
min_{λ₁,...,λ_M} E[(Ẑ(l₀) - Z(l₀))²]

s.t. Ẑ(l₀) = m_Z + Σᵢ₌₁ⁿ wᵢ (Z(lᵢ) - m_Z)
```

**权重计算**:
```
[C(l₁,l₁)  ...  C(l₁,l_M)] [w₁]   [C(l₀,l₁)]
[  ⋮       ⋱     ⋮    ] [⋮ ] = [  ⋮    ]
[C(l_M,l₁) ... C(l_M,l_M)] [w_M]   [C(l₀,l_M)]
```

**空间协方差**:
```
C(l_i, l_j) = σ_Z² R_z(l_i, l_j)
```

**预测MSE**:
```
σ_sk²(l₀) = σ_Z² - Σᵢ₌₁ᴹ wᵢ C(l₀, lᵢ)
```

**特性**: 
- 最佳线性预测器(BLP)
- 由于已知恒定均值假设而固有地无偏
- 如果过程Z是高斯的,SK在所有(线性和非线性)估计器中是全局最佳预测器

### 方法3: Trans-Gaussian Kriging

**动机**: 阴影衰落数据Z有时呈现偏态分布。高斯分布确保线性预测器在所有可能预测器(包括非线性)中的MSE最优性。

**转换方法**: 将Z转换为具有标准正态分布N(0,1)的U:
```
U(l) = CDF_N(0,1)^(-1) (CDF_Z(Z(l))) ≡ f(Z(l))
```

**逆转换与偏差校正**:

对于OK:
```
Ẑ_ok(l₀) = φ(Û_ok(l₀)) + φ''(m_U) [σ_U,ok²(l₀)/2 - μ_U]
```

对于SK:
```
Ẑ_sk(l₀) = φ(Û_sk(l₀)) + φ''(m_U)/2 [σ_U,sk²(s₀) - Σᵢ₌₁ⁿ λᵢ C(l₀, lᵢ)]
```

其中φ(·)是f(·)的逆函数,使用二阶delta方法补偿估计偏差。

### 方法4: 高斯过程回归 (GPR)

**原理**: 将Z建模为潜在随机过程Y和加性测量噪声δ_m的和:
```
Z(l) = Y(l) + δ_m
```

其中δ_m ~ N(0, σ_GP²)表示高斯噪声。

**潜在过程**:
- 零均值: m_Y = 0
- 空间协方差: C_Y(l_i, l_j) = σ_Y² R_Y(l_i, l_j)

**噪声观测的协方差**:
```
C_Z(l_i, l_j) = C_Y(l_i, l_j) + σ_GP² δ_ij
```

其中δ_ij是Kronecker delta(i=j时为1,否则为0)。

**GPR估计**:
```
Ẑ(l₀) = Σᵢ₌₁ᴹ wᵢ Z(lᵢ)
```

**权重计算**:
```
[C_Y(l₁,l₁)+σ_GP²  ...  C_Y(l₁,l_M)    ] [w₁]   [C_Y(l₀,l₁)]
[      ⋮          ⋱        ⋮          ] [⋮ ] = [   ⋮    ]
[C_Y(l_M,l₁)   ... C_Y(l_M,l_M)+σ_GP²] [w_M]   [C_Y(l₀,l_M)]
```

**预测方差**:
```
σ_GPR²(l₀) = σ_Y² + σ_GP² - Σᵢ₌₁ᴹ wᵢ C_Y(l₀, lᵢ)
```

**优势**: 
- 与SK结构相似,但通过噪声方差σ_GP²正则化
- 对噪声更鲁棒
- 明确建模测量噪声

### 方法5: MC辅助GPR(深阴影提取)

**核心原理**: 将REM分解为两个不同组件:
1. **平滑空间趋势**: 全局大尺度变化
2. **深阴影组件**: 局部高梯度SF变化

**算法流程**:

#### 阶段1: REM分解

1. **矩阵补全**:
   - 输入: M个稀疏观测(D_Z, D_ψ, D_ζ)
   - 构建5m×5m网格
   - 使用GPR估计每个网格中心: Z_ij, σ_ij
   - 核范数最小化获得平滑矩阵: Ź_mc = NuclearNormMin(Z, Σ, α)

2. **残差计算**:
   ```
   Ẑ_Δ = Z - Ź_mc
   ```

3. **深阴影识别**:
   ```
   Ẑ_ds = Z_Δ · (|Z_Δ| > T_v)
   ```
   其中T_v是阈值

4. **平滑组件**:
   ```
   Ẑ_smooth = Z - Ẑ_ds
   ```

#### 阶段2: 深阴影增强

```
Ẑ_ds = GrayscaleDilation(Ẑ_ds)
```

使用灰度膨胀操作扩展深阴影区域。

#### 阶段3: 预测

```
Ẑ = interp2(Ẑ_smooth + Ẑ_ds, ψ₀, ζ₀, 'spline')
```

使用样条插值预测目标位置。

### 约束核范数最小化

**目标**: 捕获REM的全局平滑趋势,使用核范数作为矩阵秩的代理。

**优化问题**:
```
min ||Ź||_*

s.t. |Ź_ij - Z_ij| ≤ κ σ_ij,  ∀(i,j) ∈ Ω
```

其中:
- **||·||_*** : 核范数(奇异值之和)
- **Ω**: 观测条目集合
- **κ**: 控制约束松弛的参数
- **σ_ij**: 预测标准差

**物理意义**: 强制大型传播地图中典型的平滑空间转换。

## 创新点

### 创新1: 高度依赖的三相趋势

**发现**: REM重建精度随UAV高度呈三相变化:

1. **第一阶段(h < h₁)**: 
   - 性能随高度增加而提升
   - 原因: 减少地面杂波影响

2. **第二阶段(h₁ < h < h₂)**:
   - 性能局部下降
   - 原因: 中间高度的空间相关性复杂化

3. **第三阶段(h > h₂)**:
   - 性能再次提升
   - 原因: 更稳定的传播条件

**意义**: 
- 指导UAV飞行高度选择
- 优化数据采集策略
- 揭示物理传播机制

### 创新2: 阴影衰落方差的三相趋势

**发现**: 阴影衰落方差呈非单调趋势,在两个仰角范围达到峰值:

1. **低仰角**:
   - 峰值原因: 地面级杂波
   - 特征: 多径效应显著

2. **中高仰角**:
   - 峰值原因: 天线LoS分量最强
   - 特征: 直接射线主导

**意义**:
- 理解传播环境特性
- 优化轨迹规划
- 预测重建难度

### 创新3: 带宽多样性效应

**发现**: REM重建精度随频谱带宽增加而改善。

**机制**: 频率多样性减少信号对多径衰落的脆弱性。

**实验验证**:
- 带宽范围: 1.25 MHz - 5 MHz
- 性能提升: 更宽带宽下RMSE降低
- 物理基础: 频率平均效应

**意义**:
- 指导系统设计
- 权衡带宽与精度
- 理解物理限制

### 创新4: MC辅助GPR框架

**核心创新**: 专门针对深阴影区域的重建框架。

**技术特点**:
1. **双重分解**: 平滑趋势 + 深阴影
2. **矩阵补全**: 捕获全局低秩结构
3. **形态学增强**: 灰度膨胀扩展深阴影
4. **自适应阈值**: 基于方差的置信度加权

**优势**:
- 克服矩阵补全的过平滑问题
- 保留局部高梯度特征
- 提高深阴影区域精度

### 创新5: 现场天线模式校准

**创新点**: 从实测数据估计有效天线模式,而非依赖暗室测量。

**方法**:
1. **数据驱动**: 使用实际UAV部署中的测量
2. **机身效应**: 捕获UAV结构对天线模式的真实影响
3. **混合策略**: 实测校准 + 标准增益模式(缺少数据方向)

**效果**:
- 显著提高重建精度
- 补偿电磁干扰
- 考虑机身阴影

**对比**:
- 传统方法: 暗室测量,忽略机身效应
- 本研究: 现场校准,真实部署条件

### 创新6: 全面方法比较

**系统性评估**:
- 简单Kriging (SK)
- 普通Kriging (OK)
- Trans-Gaussian Kriging (OK-TG, SK-TG)
- 高斯过程回归(GPR)
- 矩阵补全(MC)
- MC辅助GPR

**稀疏度分析**: M从10到250,覆盖极端稀疏到适度密集场景

**关键发现**:
- GPR总体表现最佳
- SK和GPR在极端稀疏下更鲁棒
- MC辅助GPR在深阴影环境优势明显

## 实验结果

### 方法性能比较

#### 总体性能排序

1. **GPR**: 最佳整体性能
2. **简单Kriging (SK)**: 极端稀疏下鲁棒
3. **普通Kriging (OK)**: 稳定但次优
4. **Trans-Gaussian Kriging**: 偏态分布下有效
5. **矩阵补全(MC)**: 单源场景可靠
6. **MC辅助GPR**: 深阴影环境最优

#### 稀疏度影响

**极端稀疏(M < 50)**:
- SK和GPR表现最稳定
- OK性能下降明显
- 深度学习方法失效

**适度稀疏(50 < M < 150)**:
- 所有方法性能改善
- GPR优势扩大
- MC辅助GPR开始发挥作用

**相对密集(M > 150)**:
- 性能趋于饱和
- 方法间差异缩小
- 深阴影区域仍有差异

### 高度影响分析

#### 三相趋势验证

**实验A1-A5 (30m - 110m)**:

**阶段1 (30m - 50m)**:
- RMSE: 从8.2 dB降至7.5 dB
- 改善: 8.5%
- 原因: 减少地面杂波

**阶段2 (50m - 90m)**:
- RMSE: 从7.5 dB升至7.8 dB
- 性能下降: 4%
- 原因: 空间相关性复杂化

**阶段3 (90m - 110m)**:
- RMSE: 从7.8 dB降至7.2 dB
- 改善: 7.7%
- 原因: 传播条件稳定

**最优高度**: 约50m或>110m

### 带宽影响分析

#### 实验C4-C5-C1 (1.25, 2.5, 5 MHz @ 40m)

**RMSE改善**:
- 1.25 MHz → 2.5 MHz: 6.2%改善
- 2.5 MHz → 5 MHz: 5.8%改善
- 总改善: 11.6%

**物理机制**:
- **频率多样性**: 不同频率经历不同衰落
- **平均效应**: 宽带测量平滑多径
- **鲁棒性**: 减少深度衰落影响

**边际效应递减**:
- 从1.25到2.5 MHz: 显著改善
- 从2.5到5 MHz: 中等改善
- 预期: 更高带宽改善有限

### 仰角影响分析

#### 阴影衰落方差趋势

**低仰角(< 20°)**:
- 方差: 高峰值
- 原因: 地面级杂波
- 特征: 树木、建筑、地形

**中仰角(20° - 60°)**:
- 方差: 相对较低
- 原因: 过渡区域
- 特征: 较稳定的传播

**高仰角(> 60°)**:
- 方差: 第二峰值
- 原因: LoS分量主导
- 特征: 直接射线强,反射影响大

**轨迹规划启示**:
- 避免极低仰角
- 中等仰角最优
- 高仰角需谨慎

### 深阴影区域性能

#### MC辅助GPR优势

**场景**: 具有显著局部阴影的环境

**对比**:
- **标准GPR**: RMSE = 9.5 dB
- **MC辅助GPR**: RMSE = 8.1 dB
- **改善**: 14.7%

**机制**:
1. **分解**: 分离平滑和深阴影组件
2. **增强**: 灰度膨胀保留高梯度
3. **融合**: 样条插值平滑合并

**适用条件**:
- 多个局部阴影区域
- 高梯度空间变化
- 足够的采样密度(M > 100)

### 天线模式校准效果

#### 实测校准 vs 暗室模式

**RMSE改善**:
- 暗室模式: 8.8 dB
- 实测校准: 7.6 dB
- **改善**: 13.6%

**校准方法**:
1. **数据收集**: 多角度实测RSRP
2. **模式估计**: 反演天线增益
3. **混合策略**: 实测 + 标准模式

**关键因素**:
- UAV机身反射
- 机载设备干扰
- 安装位置影响

## 部署效率分析

### 计算复杂度

#### Kriging方法

**OK和SK**:
- **训练**: O(M²) - 协方差矩阵计算
- **预测**: O(M) - 权重应用
- **内存**: O(M²) - 存储协方差矩阵

**适用性**:
- M < 1000: 实时可行
- M > 1000: 需要近似方法

#### GPR

**标准GPR**:
- **训练**: O(M³) - 矩阵求逆
- **预测**: O(M) - 类似Kriging
- **内存**: O(M²)

**优化策略**:
- 稀疏近似
- 诱导点方法
- 分块处理

#### MC辅助GPR

**额外开销**:
- **矩阵构建**: O(N_r × N_c) - 网格化
- **核范数最小化**: 迭代优化
- **形态学操作**: O(N_r × N_c)

**总体**: 比标准GPR慢2-3倍,但精度显著提升

### 数据需求

#### 最小采样数

**可接受性能(RMSE < 8 dB)**:
- **SK/GPR**: M ≥ 30
- **OK**: M ≥ 50
- **MC辅助GPR**: M ≥ 100

**高质量性能(RMSE < 7 dB)**:
- **GPR**: M ≥ 150
- **SK**: M ≥ 200
- **MC辅助GPR**: M ≥ 150(深阴影环境)

#### 空间分布要求

**均匀分布**: 
- 最佳性能
- 需要轨迹规划

**随机分布**:
- 性能下降10-15%
- 适用于自主采样

**聚集分布**:
- 性能下降20-30%
- 需要额外采样

### 实时性评估

#### 处理时间(100m×100m区域, M=200)

**方法对比**:
| 方法 | 训练时间 | 预测时间/点 | 总时间 |
|------|----------|-------------|--------|
| SK   | 0.5s     | 0.1ms      | 1.5s  |
| OK   | 0.6s     | 0.1ms      | 1.6s  |
| GPR  | 1.2s     | 0.1ms      | 2.2s  |
| MC-GPR | 3.5s   | 0.2ms      | 5.5s  |

**硬件**: 标准工作站(CPU: i7, RAM: 16GB)

#### 在线部署考虑

**可行场景**:
- **离线处理**: 所有方法可行
- **近实时(M < 500)**: SK, OK, GPR可行
- **实时(M < 100)**: 仅SK, OK可行

**优化方向**:
- GPU加速(2-5倍提升)
- 并行处理
- 增量更新

### 存储需求

#### 模型存储

**参数化方法**:
- **SK/OK**: 协方差参数(数KB)
- **GPR**: 核参数 + 噪声方差(数KB)
- **MC-GPR**: 网格矩阵(数MB)

**非参数化**:
- 需要存储所有训练样本
- M=200时约1MB

#### REM数据库

**分辨率**: 5m网格
**区域**: 1km × 1km
**存储**: 约200KB(单频带)

**多频带扩展**:
- 线性增长
- 压缩可减少50-70%

### 能耗考虑

#### UAV飞行能耗

**采样成本**:
- **高度**: 更高高度能耗增加10-20%
- **轨迹**: 锯齿形比直线慢2倍
- **采样点**: 悬停采样增加15%能耗

**优化策略**:
- 自适应采样(减少30-40%采样点)
- 轨迹优化(减少20-25%飞行距离)
- 高度选择(平衡精度和能耗)

#### 计算能耗

**机载处理**:
- **Kriging**: 低(嵌入式可行)
- **GPR**: 中等(需要优化)
- **MC-GPR**: 高(建议云端)

**云端处理**:
- 传输延迟: 100-500ms
- 计算延迟: 见上表
- 总延迟: < 1s(Kriging), < 3s(GPR)

## 局限性

### 方法局限性

#### 1. 单源假设

**限制**: 当前方法假设已知单个发射机位置

**影响**:
- 多源环境性能下降
- 干扰源无法建模
- 复杂场景不适用

**潜在解决**:
- 多源扩展
- 源定位算法集成
- 盲源分离

#### 2. 静态环境

**限制**: 假设环境不随时间变化

**影响**:
- 动态障碍物(车辆、人员)被忽略
- 天气影响未建模
- 时变干扰无法捕获

**潜在解决**:
- 时序建模
- 自适应更新
- 多时刻融合

#### 3. 农村环境

**限制**: 数据集和模型针对农村开阔环境

**影响**:
- 城市峡谷不适用
- 室内环境失效
- 密集植被影响未知

**潜在解决**:
- 城市数据集
- 混合传播模型
- 场景自适应

### 数据局限性

#### 1. 稀疏度下界

**限制**: M < 10时所有方法性能严重下降

**影响**:
- 极端稀疏场景不可用
- 大区域勘测成本高
- 实时更新困难

**潜在解决**:
- 迁移学习
- 物理约束增强
- 多任务学习

#### 2. 高度范围

**限制**: 实验高度限于21m - 110m

**影响**:
- 超低空(< 20m)行为未知
- 超高海拔(> 150m)未验证
- 高度外推不可靠

**潜在解决**:
- 扩展数据集
- 外推误差量化
- 物理模型融合

#### 3. 带宽限制

**限制**: 最大带宽5 MHz

**影响**:
- 宽带系统(> 10 MHz)未测试
- 5G/6G频段未知
- 超宽带不适用

**潜在解决**:
- 宽带实验
- 频率缩放研究
- 多频带融合

### 实践局限性

#### 1. 天气依赖

**限制**: 晴好天气数据

**影响**:
- 雨雪天气性能下降
- 风速影响UAV稳定性
- 温度影响电子设备

**潜在解决**:
- 天气建模
- 鲁棒性增强
- 自适应校准

#### 2. UAV类型

**限制**: 特定UAV平台(多旋翼)

**影响**:
- 固定翼UAV行为不同
- 不同尺寸UAV机身效应差异
- 通用性受限

**潜在解决**:
- 多平台验证
- 机身效应建模
- 平台自适应

#### 3. 实时性

**限制**: 大规模REM重建需要数秒

**影响**:
- 快速移动场景不适用
- 实时决策延迟
- 动态频谱管理受限

**潜在解决**:
- GPU加速
- 增量更新
- 边缘计算

### 理论局限性

#### 1. 高斯假设

**限制**: 大多数方法假设高斯过程

**影响**:
- 非高斯阴影衰落建模不准
- 极端值预测偏差
- 置信区间不可靠

**潜在解决**:
- 非参数方法
- 混合模型
- Trans-Gaussian扩展

#### 2. 平稳性假设

**限制**: 协方差函数假设空间平稳

**影响**:
- 非均匀环境性能下降
- 边界效应显著
- 局部特征丢失

**潜在解决**:
- 非平稳协方差
- 局部自适应
- 分区建模

#### 3. 线性假设

**限制**: Kriging类方法基于线性组合

**影响**:
- 非线性传播效应未捕获
- 复杂交互被忽略
- 精度上限存在

**潜在解决**:
- 非线性Kriging
- 神经网络集成
- 物理驱动学习

## 技术细节

### 协方差核函数

**指数核**:
```
k(d) = σ² exp(-d / l)
```
- **参数**: σ²(方差), l(长度尺度)
- **特性**: 平滑但不一定可微

**Matern核**:
```
k(d) = σ² (2^(1-ν) / Γ(ν)) (√(2ν) d / l)^ν K_ν(√(2ν) d / l)
```
- **参数**: ν(平滑度), l(长度尺度)
- **特性**: ν→∞时趋近高斯核

**双指数核**(本研究使用):
```
R_w(l_i, l_j) = e^(-q d_v) [a e^(-p₁ d_h) + (1-a) e^(-p₂ d_h)]
```
- **优势**: 分别建模水平和垂直相关性
- **参数**: a, p₁, p₂, q从数据学习

### 超参数估计

**最大似然估计(MLE)**:
```
θ* = argmax_θ log p(Z | θ)
```

**交叉验证**:
- K折交叉验证
- 留一法
- 网格搜索 + 贝叶斯优化

**经验贝叶斯**:
- 边际似然最大化
- 共轭梯度下降
- EM算法

### 矩阵补全算法

**核范数最小化**:
```
min ||X||_*  s.t. P_Ω(X) = P_Ω(M)
```

**奇异值阈值化**:
```
X_{k+1} = SVT_τ(Y_k)
Y_{k+1} = Y_k + δ P_Ω(M - X_{k+1})
```

**加速方法**:
- 增广拉格朗日
- 交替方向乘子法(ADMM)
- 近端梯度

### 灰度膨胀操作

**定义**:
```
(A ⊕ B)(x, y) = max_{(i,j)∈B} A(x-i, y-j)
```

**结构元素**: 3×3或5×5方形

**目的**: 扩展深阴影区域,防止过度平滑

### 插值方法

**样条插值**:
```
S(x) = Σ a_i B_i(x)
```
- **基函数**: 三次B样条
- **优势**: 平滑且局部支持
- **应用**: 最终预测步骤

**双线性插值**:
- 简单快速
- 适合初步估计
- 用于矩阵补全初始值

### 误差度量

**均方根误差(RMSE)**:
```
RMSE = √(1/N Σ |Ẑ_i - Z_i|²)
```

**平均绝对误差(MAE)**:
```
MAE = 1/N Σ |Ẑ_i - Z_i|
```

**决定系数(R²)**:
```
R² = 1 - Σ(Ẑ_i - Z_i)² / Σ(Z_i - Z̄)²
```

**95%置信区间**:
```
CI = [Ẑ - 1.96 σ, Ẑ + 1.96 σ]
```

### B. 算法伪代码

#### B.1 完整REM重建流程

```
算法: 3D REM重建
输入: 稀疏测量 {(l_i, P_Rx(l_i))}_{i=1}^M, 
      发射机位置 l_Tx, 发射功率 P_Tx,
      天线模式 G_gs, G_uav

输出: 重建REM: r̂(l) for all l

1. // 步骤1: 计算确定性路径损耗
2. for each measurement location l_i do
3.     r̂_trpl(l_i) = TRPL_Model(l_Tx, l_i, P_Tx, G_gs, G_uav)
4.     Z(l_i) = P_Rx(l_i) - r̂_trpl(l_i)
5. end for
6. 
7. // 步骤2: 估计空间相关性参数
8. θ* = EstimateCorrelation(Z, {l_i})
9. 
10. // 步骤3: 选择重建方法
11. if environment has deep shadow then
12.     method = "MC-GPR"
13. else
14.     method = "GPR"
15. end if
16. 
17. // 步骤4: 预测未采样位置
18. for each target location l do
19.     ŵ(l) = PredictShadowFading(Z, {l_i}, l, θ*, method)
20.     r̂(l) = r̂_trpl(l) + ŵ(l)
21. end for
22. 
23. return r̂(·)
```

#### B.2 MC辅助GPR详细流程

```
算法: MC辅助GPR深阴影提取
输入: 阴影衰落观测 {Z_i, ψ_i, ζ_i}_{i=1}^M,
      目标位置 (ψ_0, ζ_0), 阈值 T_v

输出: 估计阴影衰落 Ź(ψ_0, ζ_0)

1. // 阶段1: 网格化
2. d_grid = 5m
3. G_ψ = min(D_ψ):d_grid:max(D_ψ)
4. G_ζ = min(D_ζ):d_grid:max(D_ζ)
5. N_r = length(G_ψ), N_c = length(G_ζ)
6. 
7. // 阶段2: GPR预测每个网格点
8. for i = 1 to N_r do
9.     for j = 1 to N_c do
10.        [z_k, v_k] = GPR(D_Z, D_ψ, D_ζ, G_ψ(i), G_ζ(j))
11.        Z_ij = z_k
12.        Σ_ij = sqrt(v_k)
13.    end for
14. end for
15. 
16. // 阶段3: 核范数最小化
17. Ź_mc = NuclearNormMinimization(Z, Σ, α)
18. 
19. // 阶段4: REM分解
20. Z_Δ = Z - Ź_mc
21. Ź_ds = Z_Δ · (|Z_Δ| > T_v)
22. Ź_smooth = Z - Ź_ds
23. 
24. // 阶段5: 深阴影增强
25. Ź_ds = GrayscaleDilation(Ź_ds)
26. 
27. // 阶段6: 最终预测
28. Ź = interp2(Ź_smooth + Ź_ds, ψ_0, ζ_0, 'spline')
29. 
30. return Ź
```
